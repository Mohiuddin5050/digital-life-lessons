## Digital Life Lessons

<!-- ### 🎯 About  -->

Digital Life Lesson is a modern web platform where users can share meaningful life experiences, learn from others, and build a positive knowledge-sharing community. The platform supports free and premium content, community reporting, and admin moderation tools to maintain quality and safety.

---

### 🌐 Live URL

🔗 [Live Site](https://digital-life-lessions.web.app/)

---

### ✨ Key Features

- **Authentication System** 
  - Register new users with name & photo upload
  - Login / Logout functionality
  - Secure login and registration with Firebase authentication (Email/Password & Social Login).
  - Forget Password option with Gmail redirect
  
- **Create & Share Lessons**
  - Users can add life lessons with categories, emotional tones, privacy settings, and optional images.
- **Premium Access Control**
  - Premium users can create and access paid lessons, while free users can access public content.
- **Community Reporting System**
  - Users can report inappropriate lessons. Admins can review, ignore, or delete reported content.
- **Admin Dashboard**
  - Admin can manage users, review reported lessons, delete inappropriate content, and maintain platform safety.
- **Like & Favorite System**
  - Users can like lessons and track engagement.
- **Top Contributors Section**
  - Highlights creators who share the most valuable lessons.
- **Lottie Animations**
  - Interactive animations enhance user experience when adding or completing lessons.
- **My Profile Page**
  - Instantly view and update name & profile picture
- **Responsive Design**
  - Fully optimized for mobile, tablet, and desktop

### ⚙️ Technologies Used

- React.js – Frontend framework
- Firebase – Authentication & hosting
- Tailwind CSS + DaisyUI – UI styling
- React Router – Routing
- React Toastify – Notifications
- Node.js & Express.js
- Stripe (for premium features)
- React Query
- SweetAlert2
- MongoDB Atlas database
